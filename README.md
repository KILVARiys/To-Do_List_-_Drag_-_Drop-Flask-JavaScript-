# 📝 TaskFlow - Drag & Drop Task Manager

![Flask](https://img.shields.io/badge/Flask-2.3-black?style=for-the-badge&logo=flask)
![SQLite](https://img.shields.io/badge/SQLite-3-blue?style=for-the-badge&logo=sqlite)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?style=for-the-badge&logo=javascript)
![SortableJS](https://img.shields.io/badge/SortableJS-1.15-red?style=for-the-badge)

> Современный менеджер задач с возможностью перетаскивания между колонками.  
> Построен на Flask и JavaScript с использованием библиотеки SortableJS.

## ✨ Особенности

- 🎯 **Три статуса задач**: Новые, В работе, Готово
- 🖱️ **Drag & Drop**: Перетаскивайте задачи мышкой между колонками
- 📝 **Описание задач**: Добавляйте подробное описание к каждой задаче
- 💾 **Автосохранение**: Статус задачи автоматически обновляется в базе данных
- 🔍 **Модальное окно**: Просмотр полного описания по клику на задачу
- 📱 **Адаптивный дизайн**: Красиво выглядит на всех устройствах
- ⚡ **AJAX запросы**: Работа без перезагрузки страницы

## 🛠️ Технологии

### Backend
- **Flask** - Микрофреймворк для веб-приложений
- **SQLite** - Легковесная реляционная база данных
- **REST API** - Чистая архитектура API

### Frontend
- **JavaScript (ES6)** - Современный JavaScript
- **SortableJS** - Библиотека для Drag & Drop
- **Fetch API** - Асинхронные запросы к серверу
- **CSS3** - Анимации и адаптивный дизайн

## 📋 Требования

## 🔌 API Эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/tasks` | Получить все задачи |
| POST | `/api/tasks/create` | Создать новую задачу |
| POST | `/api/tasks/update` | Обновить статус задачи |

### Пример запроса на создание задачи

```json
POST /api/tasks/create
Content-Type: application/json

{
    "title": "Изучить Flask",
    "description": "Нужно разобраться с маршрутами и шаблонами"
}
```

## 🎮 Как пользоваться

1. **Создание задачи**
   - Введите название задачи
   - Добавьте описание (необязательно)
   - Нажмите "Создать задачу"

2. **Перемещение задачи**
   - Нажмите и удерживайте левую кнопку мыши на задаче
   - Перетащите в другую колонку
   - Отпустите кнопку мыши

3. **Просмотр описания**
   - Нажмите на любую задачу
   - В открывшемся окне будет полное описание
   - Закройте окно (крестик, клик вне окна или ESC)

## 🗄️ Структура базы данных

```sql
CREATE TABLE tasks (

    title TEXT NOT NULL,
    description TEXT,
    status TEXT,
    position INTEGER
);
```
